Project Resources

These are the important libraries, tools, and software used to build the AI Virtual Mouse Project.

Libraries Used:

  • OpenCV - Video processing and computer vision tasks.
  • MediaPipe - Hand tracking and landmarks detection.
  • PyAutoGUI - Mouse cursor movement and clicks control.
  • Pynput - Scrolling and advanced mouse actions.

Tools and Software:

  • Python 3.x - Programming Language.
  • Visual Studio Code - Code Editor for development.
  • Webcam - For capturing hand gestures in real-time.
