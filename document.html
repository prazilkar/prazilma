1. Introduction

This project allows you to control your computer mouse using your hand movements through webcam input and computer vision techniques.

2. Software Requirements

  • Python 3.7 or above
  • OpenCV
  • MediaPipe
  • PyAutoGUI
  • Pynput

3. Installation Steps

Install the required libraries using pip:

pip install opencv-python mediapipe pyautogui pynput

4. How It Works

The webcam captures the live video stream, which is processed to detect hand landmarks using MediaPipe. Based on finger positions, cursor movements, clicks, and scroll actions are triggered.

5. Features

  • Hand-based cursor movement
  • Single Click and Double Click using gestures
  • Scrolling with thumb and pinky finger
  • Right-click with middle and thumb finger
